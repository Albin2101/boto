###################################
AWS Key Management Service Examples
###################################

.. meta::
   :description:
   :keyword: AWS KMS code examples for the AWS SDK for Python

AWS Key Management Service (AWS KMS) is an encryption and key management service scaled for the cloud.

You can use the following examples to access AWS KMS using the AWS SDK for Python.
For more information about AWS KMS,
see the
`Amazon KMS documentation <http://aws.amazon.com/documentation/kms/>`_.
For reference information about the AWS KMS client, see the
`Client <https://boto3.readthedocs.io/en/latest/reference/services/kms.html#client>`_ class.
